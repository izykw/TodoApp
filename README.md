# TodoApp
Regular Todo app

# Technology stack
<ul>
<li>TypeScript</li>
<li>SCSS</li>
<li>React</li>
<li>Redux Toolkit</li>
</ul>

# Prerequisites
What you need to install to use the software locally
<ul>
  <li>NodeJs</li>
  <li>Npm or Yarn</li>
</ul>

# Installation
<pre>
$ npm install
$ npm start
</pre>
or if you use yarn
<pre>
$ yarn install
$ yarn start
</pre>
